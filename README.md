# Sub2API 审计管理后台

新项目独立落地目录，按项目管理规划拆分为后端、前端、文档和部署配置。

## 目录

```text
sub2api-audit-admin/
  backend/
  frontend/
  docs/
  deploy/
```

## 后端

后端基于 Laravel 12。

```bash
cd backend
php artisan --version
```

## 前端

前端基于 Vue3、Vite、TypeScript。

```bash
cd frontend
pnpm install
pnpm build
```
